# SCT_WD_4: To-Do Web App

**SkillCraft Technology Web Development Internship — Task 04**

A comprehensive task management application with LocalStorage persistence, built entirely with vanilla JS.

## Features
- **Brutalist Aesthetic:** Acid lime accents, sharp borders, and high-contrast minimal UI.
- **LocalStorage Persistence:** Your tasks are saved securely in your browser and persist across reloads.
- **Full CRUD:** Create, Read, Update, and Delete tasks seamlessly.
- **Priority & Due Dates:** Assign priorities (High/Medium/Low) and due dates/times to keep track of deadlines.
- **Filtering & Sorting:** Filter tasks by Active/Completed and sort by Creation, Due Date, or Priority.
